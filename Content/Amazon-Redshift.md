# Amazon Redshift

- Amazon Redshift is a  data warehousing service that is used for big data analytics.
- It allows you collect data from many sources and hnderstand relationships and trends across the data.
- In collaboration with Amazon Redshift Spectrum you can directly run a single SQL query against exabytes of unstructured data in data lakes.
- It allows you to achieve 10X the capacity of relational databases when it comes to BI & OLAP.
